xiami_examples
==============

xiami webapp examples . 整合了 [meteor](http://docs.meteor.com), [angular](https://angularjs.org/), [ionic](http://nionicframework.com/)

##安装

```shell

    $ cd app/client
    
    $ bower install
    
    $ cd ../..
     
    $ npm install
    
    $ npm install -g xpm2
    
    $ xpm run
